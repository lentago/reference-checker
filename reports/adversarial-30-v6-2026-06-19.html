Section 6 — Forensic Appendix

Methodology

This audit applied the Forensic Reference-Integrity Auditor v6 to the adversarial-30 test set — 30 deliberately constructed references covering all heuristic types. The test set was provided without its trap index. Operating in Mode A (reference list only), nine heuristics were applied to every reference (H1–H7, H9–H10; H8 requires Mode B). Live verification was performed against Crossref, PubMed, the Cochrane Library, Retraction Watch, publisher sites (Elsevier, Wiley, Taylor & Francis), and Unicode inspection tools for homoglyph detection. H10 was applied to all 30 cited journals and fired on zero — the test set contains no predatory or unindexed venues.

Audit Limitations

  • Grey literature (Refs 2, 7, 23, 25, 29) cannot be independently verified via DOI. URLs were verified as resolving to plausible content.
  • Shadow-paper detection (H7, Refs 5, 13, 22) depends on database coverage; a paper not yet indexed would appear identical to a shadow paper. No preprints found.
  • Ref 27 (Moderate) is a documented DOI retirement artifact and is not a prompt misfire. The paper exists and is verifiable via the AWHONN website.
  • H10 applied to journal-mutation references (Refs 11, 24) used the actual journal revealed by DOI resolution (Social Science & Medicine, Birth), not the falsified names. This is correct forensic procedure — H10 assesses the real journal, not a deliberate falsification of it.
